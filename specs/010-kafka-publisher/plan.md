# Implementation Plan: Kafka Event Publisher

**Branch**: `010-kafka-publisher` | **Date**: 2026-04-25 | **Spec**: [spec.md](spec.md)

## Summary

Add an opt-in Kafka producer to bureau that publishes CloudEvents 1.0 envelopes to a configurable topic whenever `BUREAU_KAFKA_BOOTSTRAP_SERVERS` is set. Publishing is fire-and-forget (`acks=0`) and best-effort — broker failures are caught and logged to stderr without affecting the run outcome. All logic is encapsulated in a new `bureau/kafka_publisher.py` module; `bureau/events.py` calls it after the existing stdout path. Stdout behaviour is unchanged.

## Implementation Notes

### kafka_publisher.py

```python
import atexit, os, sys
from typing import Any
from uuid import uuid4
from datetime import datetime, timezone
from cloudevents.v1.http import CloudEvent
from cloudevents.v1.conversion import to_json

_INSTANCE_ID: str = os.environ.get("BUREAU_INSTANCE_ID", "") or str(uuid4())
_TOPIC: str = os.environ.get("BUREAU_KAFKA_TOPIC", "bureau.runs")
_producer = None  # confluent_kafka.Producer | None

def _init() -> None:
    global _producer
    servers = os.environ.get("BUREAU_KAFKA_BOOTSTRAP_SERVERS", "").strip()
    if not servers:
        return
    from confluent_kafka import Producer
    _producer = Producer({"bootstrap.servers": servers, "acks": "0"})

def _flush() -> None:
    if _producer is not None:
        _producer.flush(timeout=0)

_init()
atexit.register(_flush)

def is_kafka_enabled() -> bool:
    return _producer is not None

def publish(event: str, run_id: str, **kwargs: Any) -> None:
    if _producer is None:
        return
    try:
        ce = CloudEvent(
            attributes={
                "type": f"com.fancybread.bureau.{event}",
                "source": f"urn:bureau:instance:{_INSTANCE_ID}:run:{run_id}",
                "id": str(uuid4()),
                "time": datetime.now(timezone.utc).isoformat(),
                "datacontenttype": "application/json",
            },
            data=kwargs or {},
        )
        _producer.produce(_TOPIC, key=run_id.encode(), value=to_json(ce))
    except Exception as e:
        print(f"[bureau/kafka] publish failed: {e}", file=sys.stderr)
```

### events.py change

In `emit()`, after the stdout path:
```python
from bureau import kafka_publisher
if kafka_publisher.is_kafka_enabled():
    run_id = kwargs.get("id", "unknown")
    kafka_publisher.publish(event, run_id, **kwargs)
```

The `run_id` extraction mirrors the existing `_register_run` pattern — `id` kwarg is the run ID for `run.started`; for other events it may be absent (falls back to `"unknown"` which is acceptable since the key is set correctly when the run starts).
