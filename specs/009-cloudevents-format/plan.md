# Implementation Plan: CloudEvents 1.0 Event Format

**Branch**: `009-cloudevents-format` | **Date**: 2026-04-25 | **Spec**: [spec.md](spec.md)

## Summary

Adopt CloudEvents 1.0 as an opt-in output format for bureau's `events.emit()`. When `BUREAU_OUTPUT_FORMAT=cloudevents`, every event is serialized as a single-line CloudEvents 1.0 JSON object (NDJSON) to stdout. Plain-text format (`[bureau] event  k=v`) remains the default and is unchanged. The `cloudevents` PyPI package handles envelope construction. `run.escalated` gains structured `what_happened`/`what_is_needed` fields in CloudEvents mode, replacing the current raw print block.

## Implementation Notes

### Format Selection

```python
import os
from enum import Enum

class OutputFormat(Enum):
    TEXT = "text"
    CLOUDEVENTS = "cloudevents"

_FORMAT = OutputFormat(os.environ.get("BUREAU_OUTPUT_FORMAT", "text").lower())
```

Resolved once at module import. `is_cloudevents_mode()` returns `_FORMAT == OutputFormat.CLOUDEVENTS`.

### Source URI Cache

```python
_source_uri: str = os.environ.get("BUREAU_SOURCE_URI", "urn:bureau:run:unknown")

def _register_run(run_id: str) -> None:
    global _source_uri
    if not os.environ.get("BUREAU_SOURCE_URI"):
        _source_uri = f"urn:bureau:run:{run_id}"
```

Called automatically when `run.started` is emitted (detected by event name in `emit()`).

### CloudEvents Emit

```python
from cloudevents.http import CloudEvent
from cloudevents.conversion import to_json
from datetime import datetime, timezone
from uuid import uuid4

def _emit_cloudevents(event: str, **kwargs: Any) -> None:
    ce = CloudEvent(
        attributes={
            "type": f"io.bureau.{event}",
            "source": _source_uri,
            "id": str(uuid4()),
            "time": datetime.now(timezone.utc).isoformat(),
            "datacontenttype": "application/json",
        },
        data=kwargs or {},
    )
    print(to_json(ce).decode(), flush=True)
```

### Escalation Suppression

In `escalate_node`, the raw print block is conditioned on `not events.is_cloudevents_mode()`. In CloudEvents mode, `what_happened` and `what_is_needed` are passed as kwargs to `events.emit(RUN_ESCALATED, ...)`.
